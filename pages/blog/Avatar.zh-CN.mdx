# Avatar

以前，我们想要生成特定风格的图片，通常需要依赖 LoRA 模型和提示词。然而，对于人物头像来说，如何在不改变长相的前提下实现多种艺术风格和视觉效果的转换，一直是个难题。现在，有了 InstantID，你可以轻松解锁你的创意！只需上传一张图片，通过下面分享的几个工作流模板，就能快速生成特定风格的高质量图像，例如吉卜力风格，漫画风，贴纸风等等，让你的创作变得轻松有趣。

### 1. Instant ID Basic 

这个工作流特别适合头像生成。你只需要上传一张肖像照片，它便能准确识别照片中人物的面部五官细节。通过图像处理算法，最终能够生成一张与输入照片高度相似的人物图像，并且将风格转换成独特的漫画风格。Prompt 参考：comic character. graphic illustration, comic art, graphic novel art, vibrant, highly detailed

### 2. Instant ID IP Adapter

这个工作流通过输入人物图像和特殊风格的参考图像，能够实现风格迁移的功能。例如，下图中我输入了一张线稿风格的参考图片。该工作流程会首先精确地识别输入的人物图像的面部特征，然后生成一张与输入图像面部完全相同的新图像。与此同时，生成的图像风格将会与参考的线稿图像风格一致。

### 3. Instant ID Posed 

通过输入一个人的图像和一个具有特殊姿势的参考图像，这个工作流程能够生成一张与输入图像面部完全相同的新图像，并且所生成的图像姿势将与参考图像保持一致。例如，下图中我使用了一位女士的正面照作为参考姿势。不论我输入的人物图像是什么样的姿势，最终生成的效果都会与这位女士的正面姿势保持一致。
