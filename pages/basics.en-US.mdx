# Basics

Introduces the basic usage of ComfyUI, and the basic principle of Stable Diffusion model. Upon completing this foundation, you can use ComfyUI to build basic Stable Diffusion and output various charts using Text-to-Chart method. Moreover: 

* Understand ComfyUI's various basic operations and hotkeys.
* Understand how to write Stable Diffusion prompts.
* Understand the knowledge related to LoRA.
