## 常见问题

<FAQBox title="Q1: 开源离线版 Comflowy 和在线版本 Comflowy 有什么区别？">
  目前，在线版本和开源离线版本提供相同的功能。主要区别是：

  ① 使用开源离线版本需要自行解决各种安装问题。而在线版本则可以直接使用。

  ② 在线版本使用的是我们的高性能 GPU，生图速度更快，但需要付费。离线开源版本因为使用的是你的计算机的 GPU，所以是免费的。
</FAQBox>

<FAQBox title="Q2: 我应该使用哪个版本？">
  如果你的电脑配置比较差，或者不想折腾，那么我们的云端版本可能会更适合你。你可以使用你的电脑下载安装我们的 [免费版本](https://github.com/6174/comflowyspace/releases)，并运行默认的工作流，看看生成图片的速度。并与我们的云端版本的 [速度](./blog/comflowy-cloud-faq) 进行对比，如果本地生图速度更快，那就没有必要使用我们的云端版本。
</FAQBox>

<FAQBox title="Q3: 你们提供免费试用吗？">
  不，我们不提供。

  首先，作为一家初创公司，我们目前没有资金来支付提供免费试用的费用。此外，即使我们提供试用，我们可以提供的使用量也会受到限制。对于大多数人来说，可能只能生成几张图片，这对于大多数人来说是不够的。

  因此，如果您想尝试我们的产品，我们建议下载并安装我们的开源版本，该版本永久免费，基本上提供与付费版本相同的功能，特别是在生成图像方面。

  然而，由于产品处于早期阶段，可能会有些不稳定，但我们正在不断改进。为了感谢早期支持者，我们提供折扣：Hobbyist 10% 折扣，Pro 20% 折扣。
</FAQBox>

<FAQBox title="Q4:你们提供学生折扣吗？">
  很遗憾，我们目前不提供学生折扣。对于学生，我们建议下载并安装我们的开源版本，该版本永久免费，在图像生成等功能方面与付费版本几乎相同。
</FAQBox>

<FAQBox title="Q5:什么是 GPU 积分？">
  每次运行工作流，都会消耗此 GPU 积分。每次运行消耗总积分 = GPU 每秒消耗积分数 x 运行工作流的总耗时。为了简化成本计算，每个积分的价值为 0.0001 美元，即 1/10,000 美元。
</FAQBox>

<FAQBox title="Q6:不同的 GPU 每秒消耗多少 GPU 积分？">
  T4 每秒消耗 4.8 GPU 积分，L4 每秒消耗 6.8 GPU 积分，A10G 每秒消耗 7.0 GPU 积分，A100(40G) 每秒消耗 26.1 GPU 积分，A100(80G) 每秒消耗 41.7 GPU 积分。
</FAQBox>

<FAQBox title="Q7:不同的套餐大约能生成多少张图呢？">
  能生成多少张图，取决于你使用何种 GPU，以及工作流的复杂程度。如果使用 L4 GPU 运行默认的工作流，并使用 SD1.5 模型，平均出图时间为 30 秒，Hobbyist 套餐一个月大约能生成 700 张图，Pro 套餐一个月大约能生成 2100 多张图。详细的 GPU 比较请查阅 此文档。
</FAQBox>

<FAQBox title="Q8:未使用的月度 GPU 积分会怎样？">
  目前，未使用的月度 GPU 积分不会结转到下个月。这是为了促进有效的容量规划，并确保订阅者之间的资源分配公平。
</FAQBox>

<FAQBox title="Q9:什么是仅支付活跃 GPU 使用量？">
  如果你使用云服务器（如 Google 的 Colab）运行 ComfyUI, 你会发现，即使只是在搭建工作流，也会消耗 GPU 积分。这是因为这些云服务器是按照 GPU 使用时间计费的。我们的计费方式则是按照实际使用的 GPU 积分计费，这样你就不会因为搭建工作流而浪费 GPU 积分，而只有在生图的时候，才会消耗 GPU 积分。
</FAQBox>

<FAQBox title="Q10: 我可以取消订阅吗？">
  你可以随时取消订阅，但取消将在当前计费周期结束时生效。如果你改变主意，你可以在周期结束前取消该取消订阅。
</FAQBox>

<FAQBox title="Q11: 我可以升级我的订阅计划吗？">
  你可以随时升级你的订阅计划。升级将立即生效，你将被收取两个计划之间的差额费用。
</FAQBox>

<FAQBox title="Q12: 如何添加自定义模型？">
  目前，我们不支持你手动添加自定义模型。如果你需要特别的模型，请通过 此表单 提交你的诉求，我们会将该模型添加到产品中，不过需要注意，该模型必须是在 Civitai 上可用。
</FAQBox>

<FAQBox title="Q13: 我可以安装自定义节点包吗？">
  我们不支持你安装自定义节点。但是，你可以通过 此表单 提交你想安装的插件，我们会将其手动安装到产品内。
</FAQBox>
