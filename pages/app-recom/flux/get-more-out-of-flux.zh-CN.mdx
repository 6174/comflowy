## Flux Pro

正如我们在 Flux 101 里[介绍的那样](https://flux101.com/en/basics/flux-model)，Flux 模型分为 Flux.1 Schnell 、 Flux.1 Dev 以及 Flux.1 Pro 三个版本。

前两个版本你都可以在本地使用，但 Flux.1 Pro 只能通过 API 使用，考虑到不是所有人都了解如何使用 API，所以我们还开发了一款 Flux.1 Pro 的应用供各位使用，各位可以看看 Pro 和 Dev 差异在哪里：

## Flux LoRA 

Flux 模型相对于各位常用的 Stable Diffusion 模型，或者 Midjourney 模型，最大的一个区别就在于开发者为了让模型更易于训练 LoRA，做了很多去风格化的训练。这样做也带来了一个好处，使用 Flux 模型生成的图看起来会更加真实（因为没啥风格），特别是人物的生成。

但这也导致了 Flux 模型缺少风格，生成的图片看起来，不像 Midjourney 那样好看。

所以在使用 Flux 模型生图的时候，我认为它比 Stable Diffusion 模型或者 SDXL 模型更需要使用 LoRA 模型。经过我的测试，我比较推荐各位使用以下几个 LoRA 模型。经过我的测试，我比较推荐各位使用以下几个 LoRA 模型。

### 2. Super Realism Portrait LoRA

第二个是 [Super Realism Portrait LoRA](https://huggingface.co/strangerzonehf/Flux-Super-Realism-LoRA) 。虽然 Flux 更擅长生成真实人物，但因为去掉了太多风格化的参数，所以导致其生成的人物有的时候会看起来有很浓的 AI 生成风格，而这个 LoRA 模型就是为了让 Flux 模型生成的图片看起来没有那么 AI 而设计的。

<br/>

<VerticalCard image="/app-recom/get-more-out-of-flux/realism.jpg" title="Super Realism Portrait →" description="这个应用使用了 Super Realism Portrait LoRA，如果你想要 Flux 生成的人物看起来更真实，那么这个 LoRA 模型就非常适合你。" href="https://app.comflowy.com/app/8cf707c7-e8d6-4906-9ef9-823f2b33c90d" />
